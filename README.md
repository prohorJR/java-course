ТЗ Банкомат

1. Что это за программа?
Это консольное приложение, которое работает как настоящий банкомат. Пользователь может "вставить карту" (ввести номер счета и пароль) и выполнить основные операции

2. Основные функции программы:

Вход в систему:

•Программа спрашивает номер счета

•Затем спрашивает пин-код

•Если пин-код неверный, программа сообщает об ошибке

•Главное меню: После входа появляется меню с вариантами:

•Показать баланс – программа показывает, сколько денег на счете

•Снять деньги – пользователь вводит сумму, программа проверяет, хватает ли денег, и если да – выдает их (т.е уменьшает баланс)

•Положить деньги – пользователь вводит сумму, программа добавляет ее на счет

•История операций – показывает последние транзакции по счету

•Пополнить телефон – позволяет оплатить мобильную связь

•Выйти – завершает работу с текущим счетом

3. Хранение данных:

Создать простой класс Account с полями:

•Номер счёта (id)

•PIN-код (pin)

•Баланс (balance)

•Список операций (operations)

Создать класс Operation для записи транзакций:

•Дата (date)

•Тип (type)

•Сумма (sum)

•Описание (info)

4. Требования к коду:

•Программа должна быть написана на Java

•Программа не должна ломаться при вводе неправильных данных

•История операций должна сохраняться для каждого счета

Примерные классы:

Bankomat – содержит меню и управляет программой

Account – хранит информацию об одном счете: номер, пин-код, баланс, операции

Bank – хранит массив всех счетов и умеет искать счет по номеру

Operation – хранит информацию об одной операции

OperationType – типы операций (пополнение, снятие, оплата)

6. Итог:
Программа, имитирующая главные функции банкомата: вход, проверку баланса, снятие и внесение денег с историей транзакций и пополнением баланса на номер телефона
